# Project 1: Multi-Touch Marketing Attribution & ROI Dashboard

This project is part of the Infotact Technical Internship Program - Advanced Data Analytics Project.

## Objective
The goal of this project is to analyze marketing, customer, product, transaction, and event data to support attribution analysis and ROI measurement.

## Project Files
- `Project1.ipynb` - Main analysis notebook
- `README.md` - Project overview and setup instructions
- `.gitignore` - Excludes raw datasets and unnecessary files from version control

## Datasets Used
The analysis uses the following datasets:
- customers.csv
- campaigns.csv
- transactions.csv
- products.csv
- events.csv

## Important Note
Raw dataset files are **not uploaded to this repository** as per project instructions.

## Tools Used
- Python
- Pandas
- Google Colab / Jupyter Notebook

## Key Work Done
- Loaded and explored datasets
- Checked shapes, columns, and missing values
- Converted date columns
- Filtered refunded transactions
- Merged datasets for analysis
- Previewed event-level data

## How to Run
1. Download the required datasets separately.
2. Place all dataset files in the same folder as the notebook.
3. Open `Project1.ipynb` in Google Colab or Jupyter Notebook.
4. Run all cells in order.
